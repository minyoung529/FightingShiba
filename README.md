## 게임 소개
게임의 보스인 '민영' 학생을 유니티 만점으로 만들기 위해 학생의 방해를 피해 유니티를 발사하는 슈팅 게임입니다. 

## 게임 특징
* 귀엽고 밝은 그래픽 콘셉트, 사운드 콘셉트
* 다양한 버프, 디버프 아이템
* 스킨 상점, 아이템 업그레이드 상점
